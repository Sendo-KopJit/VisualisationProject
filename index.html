Design Principle

Just a colour key
for Minkowskis3D

Every accent colour maps to a spacetime concept. Timelike intervals are blue, spacelike are red, lightlike are cyan, the same visual language across every tool, so the interface teaches physics as you explore it.

ds² = (c·Δt)² − Δx² − Δy² Minkowski interval
ds² > 0  →  TIMELIKE future cone interior
ds² < 0  →  SPACELIKE outside light cone
ds² = 0  →  LIGHTLIKE on the cone surface
τ = ∫√(ds²) / c proper time

Minkowski Spacetime Explorer

Draw worldlines in the 2D cross-section editor and explore them in interactive 3D spacetime. The reference light cone at the start event defines causal structure, segments are coloured by whether they are timelike, spacelike, or lightlike. Sixteen presets from inertial motion to Rindler acceleration, with a full observer-frame animation mode.

Open Simulation ↗
How to use
1

Open the simulation using the button above or by clicking the preview. It opens in a new tab.

2

Load a preset — click the ✎ Draw Path button at the bottom, then select the Presets tab for ready-made worldlines.

3

Or draw your own — on the x–ct tab, draw upward with your mouse to define spatial motion over time. Use y–ct to add depth.

4

Apply and start — click ✓ Apply to 3D, then press ▶ Start in the sidebar to animate the observer along the worldline.

5

Watch proper time τ accumulate on screen. Segments coloured blue are timelike, red spacelike, cyan lightlike.

6

Orbit the view by dragging the 3D canvas. Scroll to zoom. Adjust pitch and yaw in the sidebar sliders.
