Section 01 · Spec 1.1

SI Units, Quantities and Measurements

Key Definitions

SI Base UnitDefine
A unit from which all other units are derived. There are seven: kilogram, metre, second, ampere, kelvin, mole, candela. OCR wording: "A unit that cannot be expressed in terms of other units" Reject: listing derived units as examples of base units. The ampere IS a base unit; the coulomb is NOT.
Absolute uncertaintyDefine
The range above and below a measured value within which the true value is expected to lie. Written as $\pm \Delta Q$. Percentage uncertainty $= \dfrac{\Delta Q}{Q} \times 100$
Random errorDefine
Unpredictable variation in measurements that causes readings to scatter around a true value. Reduced by taking more measurements and calculating a mean. Distinguished from systematic error (which shifts all readings in the same direction and cannot be reduced by repeating).

1.1 · SI Base Units — MCQ Method

Exam method: SI base unit MCQ

  1. Memorise the seven: kg, m, s, A, K, mol, cd.
  2. Reject compound names: coulomb $= \mathrm{A\,s}$, ohm $= \mathrm{kg\,m^2\,s^{-3}\,A^{-2}}$, volt $= \mathrm{kg\,m^2\,s^{-3}\,A^{-1}}$.
The ampere IS a base unit. The coulomb is NOT — it is charge, a derived quantity.

1.2 · Percentage Uncertainty

Uncertainty rules $$\%\text{unc} = \frac{\Delta Q}{Q} \times 100$$ $$W = V \cdot I \cdot t \implies \%W = \%V + \%I + \%t$$ $$Q^n \implies \%Q^n = n\,\%Q$$ $$\text{Consistency: } \%\text{diff} = \frac{|E_\text{calc}-E_\text{ref}|}{E_\text{ref}}\times100 \quad\text{Consistent if } \%\text{diff} < \%\text{unc}$$

Exam method: W = VIt uncertainty (2024)

  1. $\%V = (0.20/12)\times100 = 1.67\%$
  2. $\%I = (1.0/80)\times100 = 1.25\%$  (note: current given in mA)
  3. $\%t = (0.01/60)\times100 = 0.017\%$
  4. Total $= 2.9\%$
Never add absolute uncertainties. Always write the explicit conclusion: "% difference < % uncertainty, therefore consistent."

1.3 · Improving Experimental Accuracy (Describe questions)

Exam method: suggest improvements to reduce uncertainty

  1. Increase the measured quantity — e.g. measure across more fringes, use a longer wire, drop from a greater height. This reduces % uncertainty without changing the absolute uncertainty.
  2. Use electronic timing — light gates, data logger, or video camera with known frame rate eliminates human reaction time error (~0.2 s).
  3. Repeat and average — reduces the effect of random errors.
  4. Control variables — state explicitly which variables are kept constant and how.
Saying "be more careful" or "reduce human error" scores 0. Name the specific technique and explain why it helps.

Section 02 · Spec 3.1

Forces and Motion

Key Definitions

Newton's First LawState
An object remains at rest or moves with constant velocity unless acted upon by a resultant force. Key phrase: "resultant force" — not just "a force". An object with balanced forces obeys Newton's 1st law.
Terminal velocityDefine
The constant velocity reached when the driving force equals the resistive forces, giving zero resultant force and therefore zero acceleration. Reject: "velocity stops increasing" without explaining the force balance. Reject: "forces cancel" — say "resultant force = 0".
Newton's Third LawState
When object A exerts a force on object B, object B exerts an equal and opposite force of the same type on object A. The forces act on different bodies. A Newton's 3rd law pair CANNOT act on the same body. Two forces on the same object in equilibrium are NOT a 3rd law pair.
MomentumDefine
The product of an object's mass and velocity. $p = mv$. A vector quantity, units kg m s$^{-1}$. Conservation of momentum: total momentum before = total momentum after, provided no external resultant force acts.

2.1 · Terminal Velocity — Explain question

Exam method: explain terminal velocity (3 marks, 2024 Q24a)

  1. Mark 1: weight = air resistance (+ upthrust if relevant).
  2. Mark 2: resultant/net force $= 0\ \mathrm{N}$.
  3. Mark 3: acceleration $= 0$, so velocity is constant.
"Forces are balanced" alone scores 0. Never say velocity decreases at terminal velocity.

2.2 · Kinematics

SUVAT equations $$v = u + at \qquad s = ut + \tfrac{1}{2}at^2 \qquad v^2 = u^2 + 2as \qquad s = \tfrac{(u+v)}{2}t$$ $$\text{Free fall: } u=0,\ a=g=9.81\ \mathrm{m\,s^{-2}} \implies v = \sqrt{2gh} \qquad g = \frac{2s}{t^2}$$

Exam method: calculate g from timing data (3 marks, 2024)

  1. Inspect all readings. If one is clearly different, state it is anomalous, then exclude it.
  2. Calculate mean $t$ from the remaining values.
  3. Substitute into $g = 2s/t^2$ and evaluate.
Silently ignoring a value loses the mark — you must identify it as anomalous first. Improvement: use electronic timer or light gate to eliminate ~0.2 s reaction time error.

2.3 · Impulse and Momentum

Impulse, force, projectile $$\text{Impulse} = F\Delta t = \Delta p = m(v_2 - v_1)$$ $$F = \frac{m(v_2-v_1)}{\Delta t} \qquad \text{Projectile: } x = v_x\,t$$
Sign rule for rebounds: if ball reverses direction, $v_1$ and $v_2$ are in opposite directions. Use a sign convention and add the magnitudes: $|\Delta p| = m(v_1 + v_2)$.

Exam method: force from wall collision + projectile check (2023 Q24)

  1. $\Delta p = m(v_2-v_1) = 0.058\times(7.2-(-3.6)) = 0.626\ \mathrm{kg\,m\,s^{-1}}$
  2. $F = \Delta p/\Delta t = 0.626/0.052 = 12\ \mathrm{N}$
  3. Horizontal range $= v_\text{rebound}\times t = 3.6\times0.58 = 2.09\ \mathrm{m}$
  4. Acceptable range: $2.0\pm0.15\ \mathrm{m}$. State: $1.85 < 2.09 < 2.15$ ✓
Forgetting the sign reversal halves $\Delta p$. The comparison to the target range must be written explicitly.

2.4 · Stokes' Law and Fluid Terminal Velocity

Stokes terminal velocity $$v = \frac{2r^2\,g\,(\rho_s - \rho_w)}{9\eta} \qquad r = \sqrt{\frac{9\eta v}{2g(\rho_s-\rho_w)}}$$

$r$ = radius (m), $\rho_s$ = particle density, $\rho_w$ = fluid density (kg m$^{-3}$), $\eta$ = viscosity (Pa s)

Exam method: show terminal velocity from displacement data (2023 Q25c)

  1. Use data from 0.2 s onwards only (before that the particle is still accelerating).
  2. Calculate $v = \Delta s/\Delta t$ for two adjacent intervals: both give $0.27\ \mathrm{m\,s^{-1}}$ → constant → terminal velocity confirmed.
Using data from 0.0 s includes the acceleration phase — the velocity is not yet constant there.

Exam method: why use a video camera over a stopwatch (2 marks, 2023 Q25a)

  1. Mark 1: time interval is very small / particle velocity is high — a stopwatch cannot resolve these intervals.
  2. Mark 2: video camera has a smaller time resolution / stopwatch has large uncertainty due to human reaction time (~0.2 s).
Saying "video camera is more accurate" without explaining why scores 0. Name the specific problem: reaction time.

2.5 · Moments and Equilibrium

Moments and power $$\text{Moment} = F \times d_\perp \qquad P = Fv = F\cos\theta\cdot v$$ $$\text{Hanging picture (2023 Q15):}\ W = 2T\cos\theta \quad (\theta\text{ from vertical})$$ $$\text{Equilibrium:}\ \sum F = 0 \text{ and } \sum M = 0$$

Exam method: explain why bending knees reduces spine force (3 marks, 2024 Q22c)

  1. Mark 1: the perpendicular distance between the line of action of the weight and the hips (pivot H) is reduced.
  2. Mark 2: therefore the moment about the hips is smaller.
  3. Mark 3: so the compressive force on the spine is reduced.
Do not reference the centre of mass changing. Do not use the knees as the pivot — the pivot is always the hips, point H.
Two-force equilibrium MCQ (2023 Q16): Only statement 1 is true (forces equal and opposite). Statement 2 is false — Newton's 3rd law pairs act on different bodies. Statement 3 is false — constant velocity also satisfies equilibrium. Answer: A (Only 1).

Section 03 · Spec 3.2

Work, Energy and Materials

Key Definitions

Young modulusDefine
The ratio of stress to strain for a material within the elastic limit. $E = \sigma/\varepsilon = (F/A)/(x/L)$. Units: Pa (N m$^{-2}$). A material property — does not depend on the dimensions of the sample.
Elastic limitDefine
The maximum stress (or force) that can be applied to a material such that it returns to its original length when the stress is removed. Beyond the elastic limit, permanent (plastic) deformation occurs.
Brittle vs DuctileDescribe
Brittle: fractures with little or no plastic deformation. Stress-strain graph is linear up to fracture with no yield point.
Ductile: undergoes significant plastic deformation before fracture. Stress-strain graph has a large plastic region after the elastic limit. Do not confuse brittleness with hardness. Glass is brittle; steel is ductile.

3.1 · Young Modulus Experiment

Young modulus from wire experiment $$\sigma = \frac{F}{A} \qquad \varepsilon = \frac{x}{L} \qquad E = \frac{\sigma}{\varepsilon} = \frac{FL}{Ax} = \frac{kL}{A}$$ $$k = \text{gradient of }F\text{-}x\text{ graph} \qquad A = \frac{\pi d^2}{4}$$

Exam method: describe how to find cross-sectional area (3 marks, 2023 Q21b)

  1. Use a micrometer (or callipers) to measure the diameter.
  2. Measure the diameter in different places along the wire and calculate a mean.
  3. Calculate $A = \pi r^2$ where $r = d/2$, or $A = \pi d^2/4$.
Must measure in multiple places — wires are not perfectly uniform. One measurement loses mark 2. Must state the formula used.

Exam method: derive E = kL/A (2 marks, 2023 Q21c-i)

  1. State all four equations: $E = \sigma/\varepsilon$, $F = kx$, $\sigma = F/A$, $\varepsilon = x/L$.
  2. Substitute clearly: $E = \dfrac{F/A}{x/L} = \dfrac{kx/A}{x/L} = \dfrac{kL}{A}$. Every step must be shown.
Do not just quote the result. OCR requires all four equations stated and clear algebraic steps.

3.2 · Springs and Elastic PE

Hooke's law and elastic energy $$F = kx \qquad E_\text{el} = \tfrac{1}{2}kx^2 = \text{area under }F\text{-}x\text{ graph}$$

Section 04 · Spec 4.1

Electricity

Key Definitions

EMF (electromotive force)Define
The energy transferred from chemical (or other) energy into electrical energy per unit charge that passes through the source. Units: V (or J C$^{-1}$). OCR exact: "energy transferred per unit charge from chemical to electrical energy". EMF = terminal pd only when current = 0. NOT the energy to the load. NOT the pd across the internal resistance. NOT "voltage of the cell".
Potential differenceDefine
The energy transferred from electrical energy to other forms per unit charge that passes between two points. Units: V (or J C$^{-1}$). Reject: "voltage across". Use "potential difference" or "pd".
ResistivityDefine
A material property that quantifies how strongly a material opposes the flow of electric current. $\rho = RA/L$. Units: $\Omega\,\mathrm{m}$. Resistivity does NOT change when you change the length or area of the wire — it is a material property. Resistance DOES change with geometry.
Internal resistanceDefine
The resistance of the source itself (battery, cell). It causes energy to be dissipated as heat inside the source, reducing the terminal pd below the EMF. "Lost volts" $= Ir$. Terminal pd $= \varepsilon - Ir$.

4.1 · Resistance and Resistivity

Resistance equations $$R = \frac{\rho L}{A} \qquad \text{Gradient of }R\text{-}L\text{ graph} = \frac{\rho}{A}$$ $$I = nAve \implies v_d = \frac{I}{nAe}$$

$n$ = number density (material property — same for all wires of the same material regardless of dimensions)

Series wires MCQ (2023 Q9): Same current $I$ and same $n$ (same material). Different $A$ means different $v_d$ and different $R$. Answer A: charge carrier density and current are equal.

4.2 · EMF and Internal Resistance

EMF and internal resistance $$\varepsilon = I(R+r) \qquad V_\text{terminal} = \varepsilon - Ir$$ $$\text{Cells in series: total EMF} = n\varepsilon,\quad r_\text{total} = nr$$

Exam method: explain why large internal resistance is undesirable (2 marks, 2024 Q21b)

  1. Mark 1: more energy is dissipated as heat in the internal resistance r (or: the pd drop $Ir$ is larger, so the terminal pd $V = \varepsilon - Ir$ is smaller).
  2. Mark 2: therefore less energy/power is delivered to the external load (bulb).
Saying "the bulb is dimmer" alone scores 0 — you must state the physical reason (energy dissipated in r). Do not reference less current; that is ignored by the mark scheme.

Exam method: show P = 3I(ε − Ir) for three cells (3 marks, 2024 Q21a)

  1. Mark 1 (B): state $\varepsilon = Ir + IR$.
  2. Mark 2 (M): three cells in series: total EMF $= 3\varepsilon$, total $r = 3r$ → $3\varepsilon = 3Ir + IR$.
  3. Mark 3 (A): $P = I^2R = I(3\varepsilon - 3Ir) = 3I(\varepsilon - Ir)$. Show every line.
Never skip steps in a "show that" question. The M mark requires seeing the total EMF and total resistance written explicitly.

4.3 · Potential Dividers and Sensing Circuits

Potential divider and LED threshold $$V_\text{out} = \frac{R_2}{R_1+R_2}\cdot V_\text{supply} \qquad eV_0 = hf \implies V_0 = \frac{hf}{e}$$

Exam method: describe R–T experiment circuit (4 marks, 2024 Q23a)

  1. Circuit diagram: battery, ammeter in series with thermistor, voltmeter in parallel with thermistor. All symbols correct.
  2. Procedure: record voltage and current readings as temperature falls from starting value, using a thermometer to measure temperature.
  3. Range: take at least 5 readings, or readings at regular temperature intervals.
  4. Analysis: calculate $R = V/I$ for each reading and plot graph of $R$ against $\theta$.
Voltmeter must be in parallel (not series). Must state at least 5 readings or regular intervals. Must say calculate R = V/I, not just "plot results".

Exam method: thermistor-LED switch-on explanation (4 marks, 2024 Q23b)

  1. Read LED switch-on voltage from I-V graph (where current just begins: e.g. $3.0\ \mathrm{V}$).
  2. Read $R_\text{therm}$ at the threshold temperature from R-$\theta$ graph (e.g. $30\ \Omega$ at $30\,^\circ\mathrm{C}$).
  3. $V_\text{LED} = \dfrac{55}{55+30}\times5 = 3.24\ \mathrm{V}$
  4. State: $3.24\ \mathrm{V} > 3.0\ \mathrm{V}$, therefore LED switches on above $30\,^\circ\mathrm{C}$.
The comparison must be written as an explicit inequality. Both the graph reading and the calculation must be shown.

Section 05 · Spec 5.1 and 5.2

Waves

Key Definitions

Longitudinal waveDefine
A wave in which the oscillations of particles are parallel to the direction of energy transfer. Examples: sound, compression waves in a spring. Reject "perpendicular" — that describes transverse waves. Must say "parallel to direction of energy transfer/propagation", not just "parallel".
Compression and RarefactionDefine
Compression: a region where molecules are closer together than average — higher pressure.
Rarefaction: a region where molecules are further apart than average — lower pressure. NOT "varying amplitude". Amplitude is fixed for a given wave; compressions are about spacing/pressure, not amplitude.
Stationary (standing) waveDefine
A wave pattern formed by the superposition of two waves of equal frequency and amplitude travelling in opposite directions. Energy is stored rather than transmitted. Has fixed nodes (zero amplitude) and antinodes (maximum amplitude). Node spacing $= \lambda/2$. Antinodes are where constructive interference occurs; nodes where destructive interference occurs.
SuperpositionDefine
When two or more waves meet at a point, the resultant displacement is the vector sum of the individual displacements at that point.
CoherenceDefine
Two sources are coherent if they have a constant phase difference and the same frequency. Required for a stable interference pattern.

5.1 · Longitudinal Waves — Explain question

Exam method: explain how compressions and rarefactions are created (3 marks, 2023 Q22a)

  1. Mark 1: molecules vibrate/oscillate.
  2. Mark 2: the oscillations are parallel to the direction of energy transfer/propagation.
  3. Mark 3: this creates regions of high and low pressure — compressions and rarefactions.
Do not reference "varying amplitude". Do not say oscillations are perpendicular. Must say "parallel to direction of energy transfer" — not just "parallel".

5.2 · Stationary Waves — Formation and Imperfect Nodes

Stationary wave relationships $$\text{Node spacing} = \frac{\lambda}{2} \implies \lambda = 2d_\text{node}$$ $$v = f\lambda = 2f\,d_\text{node} \qquad \text{Example: }d = 0.84\ \mathrm{m},\ f = 200\ \mathrm{Hz} \implies v = 336\ \mathrm{m\,s^{-1}}$$

Exam method: explain stationary wave formation with nodes and antinodes (3 marks, 2023 Q22b-i)

  1. Mark 1: the incident and reflected waves interfere / superpose.
  2. Mark 2: constructive interference (waves in phase) gives maximum amplitude — these are antinodes.
  3. Mark 3: destructive interference (waves in antiphase) gives minimum/zero amplitude — these are nodes.
Do not write "superimpose" — OCR ignores this term. Use "superpose" or "interfere". Antinodes are constructive; nodes are destructive — do not mix these up.

Exam method: explain why nodes are not exactly zero (3 marks, 2023 Q22b-ii)

  1. Mark 1: the amplitude/intensity/energy of the wave decreases with distance from the source.
  2. Mark 2: the reflected wave has a lower amplitude than the incident wave.
  3. Mark 3: incomplete destructive interference occurs — the amplitudes do not fully cancel, so nodes are not exactly zero. The difference between amplitudes increases with distance from the reflector, so nodes further from A are larger.
Saying "energy is absorbed" alone scores 0 unless you link it to the amplitude difference causing incomplete cancellation.

Exam method: suggest improvement to reduce uncertainty in speed of sound (1 mark, 2023 Q22b-iv)

  • Measure the distance across more than one node spacing (e.g. across 5 gaps) and divide — reduces % uncertainty in distance.
  • Or: repeat measurements and calculate a mean.
  • Or: use a lower frequency (gives a longer wavelength, so larger node spacing → smaller % uncertainty).

5.3 · Young Double-Slit and Interference

Young's double-slit and interference conditions $$w = \frac{\lambda D}{d} \implies \lambda = \frac{wd}{D} \qquad \text{If "distance to }n\text{th fringe" given: } w = \frac{\text{total}}{n}$$ $$\text{Constructive: path diff} = n\lambda \qquad \text{Destructive: path diff} = (n+\tfrac{1}{2})\lambda$$
What Young's experiment demonstrates: diffraction (at each slit), superposition, and the wave nature of light. It does NOT demonstrate refraction. MCQ asking which property is NOT demonstrated: always refraction.

5.4 · Polarisation — Two MCQ Types

Exam method: distinguish the two polarisation MCQ types

  1. Type 1 — unpolarised source, filter then rotated (2023 Q13): After the filter halves the intensity by selecting one polarisation plane, rotating the filter does not change the intensity — it still transmits half the unpolarised light. Answer: D (does not change).
  2. Type 2 — polarised source, e.g. laptop screen (2024 Q4): At 90° the filter is perpendicular to the polarisation plane → dark. At 180° it realigns → full brightness. Sequence: normal → dark → normal. Answer: B.
Read carefully: is the source already polarised (Type 2), or is the source unpolarised and only the filter is polarising (Type 1)?

5.5 · Refraction and TIR

Snell's law and critical angle $$n_1\sin\theta_1 = n_2\sin\theta_2 \qquad \sin\theta_c = \frac{n_2}{n_1}\ (n_1 > n_2)$$ $$n = 1.5 \implies \theta_c = 41.8^\circ \qquad \text{If incidence} > \theta_c \implies \text{TIR}$$

Section 06 · Spec 6.1

Quantum Physics and the Photoelectric Effect

Key Definitions

PhotonDefine
A discrete packet (quantum) of electromagnetic energy. The energy of a photon is $E = hf$, where $h$ is the Planck constant and $f$ is the frequency of the radiation. Reject: "particle of light" without mentioning energy or frequency.
Work function $\phi$Define
The minimum energy needed to release an electron from the surface of a metal. $\phi = hf_0$ where $f_0$ is the threshold frequency. NOT "the energy needed to remove an electron from an atom". The electron is already free in the metal — the work function is the energy to escape the surface.
Threshold frequency $f_0$Define
The minimum frequency of electromagnetic radiation required to release an electron from the surface of a given metal. Below this frequency, no electrons are emitted regardless of intensity.
$KE_\text{max}$State meaning
The maximum kinetic energy of an electron emitted from the surface of a metal due to the photoelectric effect. It depends on frequency only, not intensity. From Einstein's equation: $KE_\text{max} = hf - \phi$

6.1 · Photoelectric Effect — Explain questions

Photoelectric equations $$hf = \phi + KE_\text{max} \qquad h = 6.63\times10^{-34}\ \mathrm{J\,s}$$ $$\text{Threshold: }hf_0 = \phi \qquad W = eV \qquad I = \frac{P}{A} = \frac{\dot{n}\,hf}{A}$$
Intensity vs frequency summary: Increasing intensity → more photons/s → more electrons/s → higher current. $KE_\text{max}$ unchanged. Increasing frequency → higher photon energy → higher $KE_\text{max}$. Rate of emission unchanged (if intensity unchanged).

Exam method: explain why frequency must exceed a minimum value (3 marks, 2023 Q23b)

  1. Mark 1: light is made of discrete packets of energy called photons.
  2. Mark 2: an electron is only emitted if the energy of a single photon is greater than or equal to the work function.
  3. Mark 3: the energy of a photon is proportional to its frequency ($E = hf$), so the frequency must exceed the threshold value $f_0 = \phi/h$.
Do not mention intensity — that is irrelevant here. The problem is energy per photon, not the rate of photon arrival. One photon interacts with one electron (one-to-one).

Exam method: explain why current is proportional to intensity (3 marks, 2023 Q23c-i)

  1. Mark 1: the intensity of radiation is proportional to the rate of incident photons (above threshold frequency).
  2. Mark 2: each photon interacts with one electron (one-to-one interaction) — so more photons per second means more electrons emitted per second.
  3. Mark 3: current $=$ rate of flow of charge $= \dot{n}_e \cdot e$, so current is proportional to the rate of electron emission, which is proportional to intensity.

Exam method: explain why below threshold frequency leaf does not fall (2 marks, 2024 Q25c)

  1. Mark 1: the energy of a photon is less than the work function of the metal.
  2. Mark 2: the electron cannot receive enough energy from a single photon to be emitted from the surface.
Do not say "the radiation is not intense enough" — intensity is irrelevant here. The problem is photon energy, not photon rate.

Exam method: gold leaf electroscope falls (3 marks, 2024 Q25b)

  1. Mark 1: electrons on the metal surface absorb photons in a one-to-one interaction.
  2. Mark 2: electrons are emitted from the metal plate.
  3. Mark 3: the electroscope loses negative charge — so the leaf becomes less charged and falls.
Do not say the leaf gains positive charge — electrons leave, it does not gain positive charges. Electrons come from the metal plate, not the gold leaf itself.

Section 07 · Written Answer Technique

Written Answer and 6-Mark Technique

7.1 · General Written Answer Principles

Structure for all explain/describe questions

1State the physical law or mechanism first. E.g. "The resultant force on the feather is zero because weight equals air resistance."
2Progress logically: cause → effect → consequence. Each sentence should follow from the last. OCR mark schemes are written as a chain — match that chain.
3Use precise OCR vocabulary. "Resultant force" not "net force". "Rate of flow of charge" not "current flowing". "Perpendicular distance" not "distance". "Superpose" not "superimpose".
4Quantify where possible. "Increases by a factor of 4" is better than "increases". "Zero resultant force" is better than "balanced".
5End with the observable outcome explicitly linked back to the opening mechanism. This closes the argument and picks up the A mark.

7.2 · Level of Response (LoR) — 6-Mark Questions

LoR questions are marked holistically. The mark scheme gives level descriptors and a communication statement. Science content determines the level; quality of communication determines whether you get the higher or lower mark within that level.

Generic LoR Level Descriptors 6 marks
Level 3
5–6
A comprehensive and logically sequenced account, correct throughout, covering all the major points. Links between ideas are clear and the argument is coherent.
Communication statement: well-structured prose, correct spelling and grammar throughout, appropriate use of technical vocabulary.
Level 2
3–4
A mostly correct account covering several major points. Some logical sequencing but gaps in the argument or some minor errors. Links between ideas are present but not always explicit.
Communication statement: generally clear prose, mostly correct technical vocabulary, minor errors in spelling or grammar.
Level 1
1–2
Some relevant points included but significant gaps, errors or misunderstandings. Little logical structure. Isolated correct statements without a coherent argument.
Communication statement: limited structure, vocabulary sometimes imprecise, errors in spelling or grammar may be frequent.
To reach Level 3: cover all three main mark points AND sequence them logically AND use correct technical vocabulary. Hitting two of these three usually gives Level 2.

7.3 · LoR Model Answer Frameworks

These frameworks match the type of 6-mark questions that have appeared on H156/01. Use them as templates.

Framework: Explain a physical phenomenon from first principles e.g. photoelectric effect, stationary waves
L3 content
Opening: State the underlying physical law or model (e.g. "Light consists of photons, each with energy $E = hf$").
Middle: Apply the model to the situation step by step, naming each quantity (e.g. "For emission, the photon energy must exceed the work function $\phi$, so $hf \geq \phi$").
Consequence: State what is observed and why (e.g. "Below the threshold frequency no electrons are emitted regardless of intensity because no single photon carries sufficient energy").
Extension: Address the other variable (e.g. "Increasing intensity increases the photon rate and therefore the emission rate and current, but does not change $KE_\text{max}$").
Always name the quantities with symbols and link them with causal language: "because", "therefore", "this means that".
Framework: Describe an experimental method e.g. measure resistivity, find Young modulus
L3 content
Apparatus: Name all instruments with their function (micrometer for diameter, metre rule for length, ammeter in series, voltmeter in parallel).
Procedure: State what is varied, what is measured, and how (e.g. "Add masses in 100 g increments, record extension using a ruler and set square at eye level to avoid parallax").
Range and repeats: State the number of readings (at least 5–6 over a suitable range) and that readings are repeated to reduce random error.
Analysis: State exactly how the target quantity is calculated from the measurements, including which graph to plot and what the gradient represents.
Marks are often awarded for: correct instrument named (1), correct circuit/setup (1), procedure detail (1), range/repeats (1), analysis (1), graph/gradient (1).

7.4 · Precision Vocabulary Pairs

Use these exact phrases — OCR mark schemes ALLOW these and often REJECT informal alternatives:

✓ "resultant force" — not "net force" or "total force"
✓ "rate of flow of charge" — not "current flowing"
✓ "perpendicular distance from the pivot" — not "distance from the pivot"
✓ "superpose" or "interfere" — not "superimpose" (ignored by OCR)
✓ "work function" — not "energy needed to remove an electron"
✓ "one-to-one interaction between photon and electron" — not "the electron absorbs the photon" alone
✓ "amplitude decreases with distance from source" — not "the wave gets weaker"
✓ "incomplete destructive interference" — not "they don't cancel properly"
✓ "terminal pd" — not "voltage of the battery"
✓ "energy dissipated as heat in the internal resistance" — not "energy wasted inside the battery"

EMF$\varepsilon = I(R+r)$. $V = \varepsilon - Ir$. Lost volts $= Ir$. EMF $=$ terminal pd when $I = 0$
Momentum$p = mv$. Impulse $= F\Delta t = \Delta p = m(v_2-v_1)$. Rebound: add speeds for $|\Delta p|$
Kinematics$v=u+at$. $s=ut+\frac{1}{2}at^2$. $v^2=u^2+2as$. Free fall: $v=\sqrt{2gh}$, $g=2s/t^2$
Young modulus$E = \sigma/\varepsilon = FL/(Ax) = kL/A$. $\sigma=F/A$, $\varepsilon=x/L$, $k=$ gradient of $F$-$x$ graph
Springs$F=kx$. $E_\text{el}=\frac{1}{2}kx^2=$ area under $F$-$x$ graph
Waves$v=f\lambda$. Young: $w=\lambda D/d$. Node spacing $=\lambda/2$. $n_1\sin\theta_1=n_2\sin\theta_2$
Sound (stationary)$v = 2f\,d_\text{node}$. Improve: measure across multiple node spacings
Quantum$E=hf$. $hf=\phi+KE_\text{max}$. $eV_0=hf$. Intensity $= P/A = \dot{n}hf/A$
Stokes' law$v=\dfrac{2r^2g(\rho_s-\rho_w)}{9\eta}$. Rearranged: $r=\sqrt{\dfrac{9\eta v}{2g(\rho_s-\rho_w)}}$
Uncertainty$\%\text{unc}=(\Delta Q/Q)\times100$. Multiply/divide: add $\%$. $Q^n$: multiply by $n$
Consistency$\%\text{diff}=|E_\text{calc}-E_\text{ref}|/E_\text{ref}\times100$. Consistent if $\%\text{diff}<\%\text{unc}$
Charge$Q=ne$. $e=1.6\times10^{-19}\ \mathrm{C}$. Lost electrons $\Leftrightarrow$ gained positive charge
Equilibrium$W=2T\cos\theta$ (cable at $\theta$ from vertical). $\sum F=0$, $\sum M=0$
